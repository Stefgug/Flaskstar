Project overview

From the RAGstar reference implementation.

View repository

RAGstar is a small app that helps you find the right GitHub repository by asking a natural language question. It fetches repository content, generates short summaries with Ollama, stores them in ChromaDB, and lets you search using a hybrid score that blends dense embeddings with BM25.

Core flow

  • Fetches README.md content with gitingest.
  • Summarizes repositories using a local Ollama model.
  • Indexes summaries in ChromaDB with sentence-transformer embeddings.
  • Answers queries with hybrid dense + BM25 scoring.
